1. Download repo
```bash
git clone https://github.com/Vorobaa/Flask.git
```
2. Create venv and connect him to pyCharm
```bash
python -m venv ./venv
```
3. Install requirements
```bash
pip install -r .\requirements.txt
```
4. Start project
```bash
python -m flask --app board run --port 8000 --debug
```
